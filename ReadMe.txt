1) "Linked_list.cs" Реализация односвязного списка на C# в виде класса с 
функциями добавления и удаления элементов, сортировкой, записью 
и чтением из файла.
2) "linked_list.cpp" Реализация односвязного списка на C++ элементов "trail"
содержащих поля "начало пути", "конец пути", "номер пути" функциями добавления
и удаления элементов, сортировкой.
3) "sorting.cpp" Реализация базовых сортировок на C++.
4) "vector_operations.cpp" Реализация векторов и операций над ними в виде класса.
